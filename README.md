# Proyecto-FInal-POO
SIGE: Sistema Integral de Gestión Ecuestre 🐎. App de escritorio en Python (CustomTkinter) para administrar centros ecuestres. Incluye historial médico auto-calculable, control financiero, inventario, mapa interactivo de caballerizas y directorio CRM. Arquitectura basada en POO con persistencia de datos local en JSON.
