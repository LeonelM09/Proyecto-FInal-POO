**SIGE: Sistema Integral de Gestión Ecuestre** 

SIGE es una solución tecnológica desarrollada en Python diseñada para automatizar y centralizar la administración operativa, clínica y financiera de instalaciones ecuestres. El sistema permite pasar de registros manuales a una gestión digital eficiente, facilitando la toma de decisiones estratégicas mediante el análisis de datos.

**Alcance de la Herramienta**

**Gestión Clínica**

• Bitácora Médica: Seguimiento de protocolos preventivos como herrajes y vacunas.

• Sistema de Semáforo: Indicadores visuales sobre el estado de salud y vencimientos.

• Generación de Recetas: Exportación de raciones y tratamientos en archivos de texto.


**Control Operativo y de Inventario**

• Mapa de Caballerizas: Grid dinámico para la asignación y monitoreo de boxes.

• Gestión de Stock: Control de suministros con alertas automáticas de existencias críticas (≤ 5 unidades).

• Seguimiento de Entrenamiento: Registro de asistencia y progreso por etapas (Iniciación a Avanzado).


**Arquitectura Técnica**

• Interfaz: Desarrollada con CustomTkinter para una experiencia de usuario moderna.

• Lógica de Negocio: Implementación de Programación Orientada a Objetos (POO) en el backend para un código escalable.

• Análisis de Datos: Uso de Pandas y Matplotlib para el procesamiento y visualización estadística.

• Persistencia: Almacenamiento local mediante estructuras JSON.
